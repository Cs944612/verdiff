# Code of Conduct

## Our Pledge

We want `verdiff` to be a welcoming, respectful project for users and contributors.

## Our Standards

- Be respectful and constructive.
- Assume good intent.
- Focus criticism on ideas and implementation, not people.
- Share benchmarks and evidence when discussing performance tradeoffs.

## Enforcement

Project maintainers may remove or reject contributions that violate this code of conduct.
